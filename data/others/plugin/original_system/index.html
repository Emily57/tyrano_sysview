これは、自分で追加実装した original_system です。
[sysview]を使えばセーブなどで読み込むHTMLを切り替えできますが、
tyrano.plugin.kag.menu と [sysview] を組み合わせれば、存在しない全く新しい機能も作れます。
例えば、インフォメーションの表示などに便利かもしれません。
詳しくは以下のファイルのコードをチェックしてください。

data/others/plugin/original_system/index.html
→ 今見ている画面を作ってるHTML
tyrano/plugins/kag/kag.menu.js
→ displayOriginalSystem を定義。
→ 関数名はなんでもいいけど、display + [sysview] で使う名前をキャメルケースにした名前が無難でしょう。
tyrano/plugins/kag/kag.tag.js
→ original_system で displayOriginalSystem を呼び出すよう定義
